# Group Photos

Put real Rhythm Without Borders photos in this folder.

Recommended filenames:

```text
hero-group.jpg
circle-1.jpg
circle-2.jpg
```

Use `hero-group.jpg` for the main top image. Landscape photos work best, ideally around 1600 px wide or larger.

Tips:

- Prefer `.jpg` for photos.
- Keep each image under about 1-2 MB if possible.
- Avoid spaces in filenames.
- Make sure everyone in the photo is okay with it being public on the website.
